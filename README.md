# scaffold
A project scaffold for Python
## Step 1
Type this in your virtual environment to get an SSH Key  
```
ssh-keygen -t rsa  
cat /home/ec2-user//.ssh/id_rsa.pub
# the string is within the public key section of the image below 
```
